# AGENTS.md

## Startup Context (Required Every Session)
Read these files first:
1. AGENT_GUIDE.md
2. ../CLAUDE.md if present
3. CLAUDE.md
4. log.md
5. MEMORY.local.md if present and relevant to the task
6. MEMORY.md if present and relevant to the task
7. the current or most recent memory/YYYY-MM-DD.md file when continuing recent work

## Workflow
1. Follow AGENT_GUIDE.md shared rules.
2. Apply parent context, then folder-specific constraints.
3. Update log.md while working.
4. Update CLAUDE.md when patterns are confirmed.
5. Promote durable notes to MEMORY.md or MEMORY.local.md at the bounded cadence defined in AGENT_GUIDE.md.

## Session Closeout
Before ending a substantial session:
1. Update `log.md`.
2. Promote durable memory if warranted.
3. Update `CLAUDE.md` only for confirmed durable patterns.

## Priority
1. Runtime system/developer instructions
2. This AGENTS.md
3. AGENT_GUIDE.md
4. ../CLAUDE.md if present
5. CLAUDE.md
6. User request
